# SentraAI

A comprehensive visitor management and access control system with AI-powered features.

## Project Structure

- **Client**: React-based frontend application
- **Server**: Express.js backend with AI service integration

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. **Server Setup**
   ```bash
   cd server
   npm install
   ```

2. **Client Setup**
   ```bash
   cd client
   npm install
   ```

### Running the Application

**Server**
```bash
cd server
npm start
```

**Client**
```bash
cd client
npm run dev
```

## Features

- Visitor entry management
- Access control and approval system
- Alert notifications
- Admin dashboard
- Automatic logging
- AI-powered analytics
